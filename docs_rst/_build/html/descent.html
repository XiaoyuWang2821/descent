descent.main module¶

Main routines for the descent package

descent.main.optimize¶

Main optimization loop

Parameters:

algorithm : function

A function which returns a generator that yields new iterates in a descent sequence (for example, any of the other functions in this module)

f_df : function

A function which takes one parameter (a numpy.ndarray of parameters) and returns the objective and gradient at that location

x0 : array_like

A numpy array consisting of the initial parameters

callbacks : list, optional

A list of functions, each which takes one parameter (a dictionary containing metadata). These functions should have side effects, for example, they can log the parameters or update a plot with the current objective value. Called at each iteration.

maxiter : int, optional

The maximum number of iterations (Default: 1000)

minibatches : list, optional

Used for minibatch optimization. An optional list of data (req)
